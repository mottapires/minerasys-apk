# MineraSys App - Flutter

App mobile para controle de areal.

## Como compilar no Codemagic

1. Faça upload deste projeto no GitHub
2. Acesse https://codemagic.io
3. Conecte seu repositório
4. Clique em "Start new build"
5. Aguarde 10-15 minutos
6. Baixe o APK gerado

## APIs necessárias

- https://areal.smgengenharia.com.br/api/configuracoes.php
- https://areal.smgengenharia.com.br/api/apontador-sync.php
- https://areal.smgengenharia.com.br/api/operador-sync.php
